Transform Function Descriptions

You can modify and manipulate worksheet data by entering SigmaPlot's extensive mathematical transformation language into the User-Defined Transform dialog box. Type transform instructions into the Edit Transform field. You can enter up to 32,000 characters.

The following list groups transforms by function type. It is followed by an alphabetical reference containing complete descriptions of all transform functions and their syntax, with examples.

Worksheet Functions

These worksheet functions are used to specify cells and columns from the worksheet, either to read data from the worksheet for transformation, or to specify a destination for transform results.

Function Description
block The block function returns a specified block of cells from the worksheet. block
blockheight, blockwidth The blockheight and blockwidth functions return a specified block of cells or block dimension from the worksheet. blockheight, blockwidth
cell The cell function returns a specific cell from the worksheet. cell
col The col function returns a worksheet column or portion of a column. col
put into The put into function places variable or equation results in a worksheet column. put into
subblock The subblock function returns a specified block of cells from within another block. subblock

Data Manipulation Functions

The data manipulation functions are used to generate non-random data, and to sample, select, and sort data.

Function Description
data The data function generates serial data. data
if The if function conditionally selects between two data sets. if
nth The nth function returns an incremental sampling of data. nth
sort The sort function rearranges data in ascending order. sort

Matrix Functions

These functions compute common operations for matrices, as well as providing solutions to systems of linear equations, multiple linear regression problems, and the eigenvalue problem for symmetric matrices.

Function Description
inv The inv function generates the inverse matrix of an invertible square matrix provided as a block. inv
trp The trp function generates the transpose of any matrix provided as a block. The entries of the matrix can be non-numeric.
prod The prod function computes the product of two matrices where each matrix is provided as a block or a range. The product is defined as usual matrix multiplication where entries in the result are computed as the inner products of rows of the first matrix and columns of the second matrix.
eigen The eigen function computes the eigenvalues and corresponding eigenvectors for a real symmetric matrix that is provided as a block.
linsys The linsys function solves systems of linear equations and multiple linear regression problems.

Trigonometric Functions

SigmaPlot provides a complete set of trigonometric functions.

Function Description
arccos This functions returns the arccosine, of the specified argument. arccos
arcsin This functions returns the arcsine of the specified argument. arcsin
arctan This functions returns the arctangent of the specified argument. arctan
cos This function returns the cosine of the specified argument. cos
sin This function returns the sine of the specified argument. sin
tan This function returns the tangent of the specified argument. tan
cosh This function returns the hyperbolic cosine of the specified argument. cosh
sinh This function returns the hyperbolic sine of the specified argument. sinh
tanh This function returns the hyperbolic tangent of the specified argument. tanh

Numeric Functions

The numeric functions perform a specific type of calculation on a number or range of numbers and returns the appropriate results.

Function Description
abs The abs function returns the absolute value. abs
exp The exp function returns the values for e raised to the specified numbers. exp
factorial The factorial function returns the factorial for each specified number. factorial
mod The mod function returns the modulus, or remainder of division, for specified numerators and divisors. mod
ln The ln function returns the natural logarithm for the specified numbers. ln
log The log function returns the base 10 logarithm for the specified numbers. log
sqrt The sqrt function returns the square root for the specified numbers. sqrt

Range Functions

The following functions give information on ranges.

Function Description
count The count function returns the number of numeric values in a range. count
makeblock The makeblock function returns a range or block of data having a specified size using a range of data that you supply. makeblock
missing The missing function returns the number of missing values and text strings in a range. missing
size The size function returns the number of data points in a range, including all numbers, missing values, and text strings.size

Accumulation Functions

The accumulation functions return values equal to the accumulated operation of the function.

Function Description
diff The diff function returns the differences of the numbers in a range. diff
sum The sum function returns the cumulative sum of a range of numbers. sum
total The total function returns the value of the total sum of a range. total

Random Generation Functions

You can use the two random number generating functions to create a series of normally or uniformly distributed numbers.

Function Description
gaussian The Gaussian function is used to generate a series of normally (Gaussian or bell shaped) distributed numbers with a specified mean and standard deviation. gaussian
random The random function is used to generate a series of uniformly distributed numbers within a specified range. random

Precision Functions

Use the precision functions to convert numbers to whole numbers or to round off numbers.

Function Description
int The int function converts numbers to integers. int
prec The prec function rounds numbers off to a specified number of significant digits. prec
round The round function rounds numbers off to a specified number of decimal places. round

Statistical Functions

The statistical functions perform statistical calculations on a range or ranges of numbers.

Function Description
avg The avg function calculates the averages of corresponding numbers across ranges. It can be used to calculate the average across rows for worksheet columns. avg
max The max function returns the largest value in a range. max
min The min function returns the smallest value in a range. min
mean The mean function calculates the mean of a range. mean
median The median function calculates the median of a range. median
runavg The runavg function produces a range of running averages. runavg
stddev The stddev function returns the standard deviation of a range. stddev
stderr The stderr function calculates the standard error of a range. stderr

Area and Distance Functions

Use these functions to calculate the areas and distances specified by X,Y coordinates. Units are based on the units used for X and Y.

Function Description
area The area function finds the area of a polygon described in X,Y coordinates. area
dist The distance function calculates the distance of a line whose segments are described in X,Y coordinates. dist
partdist The partdist function calculates the distances from an initial X,Y coordinate to successive X,Y coordinates in a cumulative fashion. partdist

Curve Fitting Functions

These functions are designed to be used in conjunction with SigmaPlots nonlinear curve fitter, to allow automatic determination of initial equation parameter estimates from the source data. You can use these functions to develop your own parameter determination function by using the functions provided with the Standard Regression Equations library provided with SigmaPlot.

Function Description
ape This function is used for the polynomials, rational polynomials and other functions which can be expressed as linear functions of the parameters. A linear least squares estimation procedure is used to obtain the parameter estimates. ape
dsinp This function returns an estimate of the phase in radians of damped sine functions. dsinp
fwhm This function returns the x width of a peak at half the peaks maximum value for peak shaped functions. fwhm
lowess The lowess algorithm is used to smooth noisy data. Lowess means locally weighted regression. Each point along the smooth curve is obtained from a regression of data points close to the curve point with the closest points more heavily weighted.lowess
lowpass The lowpass function returns smoothed y values from ranges of x and y variables, using an optional user-defined smoothing factor that uses FFT and IFFT. lowpass
sinp This function returns an estimate of the phase in radians of sinusoidal functions.sinp
x25 This function returns the x value for the y value 25% of the distance from the minimum to the maximum of smoothed data for sigmoidal shaped functions. x25
x50 This function returns the x value for the y value 50% of the distance from the minimum to the maximum of smoothed data for sigmoidal shaped functions. x50
x75 This function returns the x value for the y value 75% of the distance from the minimum to the maximum of smoothed data for sigmoidal shaped functions. x75
xatymax This function returns the x value for the maximum y in the range of y coordinates for peak shaped functions. xatymax
xwtr This function returns x75-x25 for sigmoidal shaped functions. xwtr

Fast Fourier Transform Functions

Use these functions to remove noise from and smooth data using frequency-based filtering.

Function Description
fft The fft function finds the frequency domain representation of your data. fft
invfft The invfft function takes the inverse fft of the data produced by the fft to restore the data to its new filtered form. invfft
real The real function strips the real numbers out of a range of complex numbers. real
img The img function strips the imaginary numbers out of a range of complex numbers. imaginary (img)
complex The complex function converts a block of real and/or imaginary numbers into a range of complex numbers. complex
mulcpx The mulcpx function multiplies two ranges of complex numbers together. mulcpx
invcpx The invcpx takes the reciprocal of a range of complex numbers. invcpx

Special Constructs

Transform constructs are special structures that allow more complex procedures than functions.

Function Description
for The for statement is a looping construct used for iterative processing. for
if...then...else The if...then...else construct proceeds along one of two possible series of procedures based on the results of a specified condition. if...then...else

Miscellaneous Functions

These functions are specialized functions which perform a variety of operations.

Function Description
choose The choose function is the mathematical n choose r function. choose
histogram The histogram function generates a histogram from a range or column of data. Creating Histograms
implicit

The implicit function finds the values of a function defined implicitly by an equation that includes one or more independent variables, each have a specified range of values. implicit
interpolate The interpolate function performs linear interpolation between X,Y coordinates. interpolate
lookup The lookup function compares values with a specified table of boundaries and returns either a corresponding index from a one-dimensional table, or a corresponding value from a two-dimensional table. lookup
polynomial The polynomial function returns results for specified independent variables for a specified polynomial equation. polynomial
rgbcolor The rgbcolor(r,g,b) color function takes arguments r,g, and b between 0 and 255 and returns color to cells in the worksheet. rgbcolor
root

The root function finds the roots or singularities of one or more functions of one variable over a finite interval and returns the range of values found. root
