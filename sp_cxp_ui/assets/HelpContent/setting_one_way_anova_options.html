Setting One Way ANOVA Options

  1. Select One Way ANOVA from the Select Test drop-down list in the SigmaStat group on the Analysis tab.
  2. Click Current Test Options. The Options for One Way ANOVA dialog box appears with three tabs:
    • Assumption Checking. Adjust the parameters of a test to relax or restrict the testing of your data for normality and equal variance.
    • Results. Display the statistics summary for the data in the report and save residuals to a worksheet column.
    • Post Hoc Test. Compute the power or sensitivity of the test and enable multiple comparisons.
    Tip: If you are going to run the test after changing test options, and want to select your data before you run the test, drag the pointer over your data.
  3. To continue the test, click Run Test.
  4. To accept the current settings and close the options dialog box, click OK.
