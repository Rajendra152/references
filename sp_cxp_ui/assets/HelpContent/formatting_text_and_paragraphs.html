Formatting Text and Paragraphs

To view or hide report properties, select Report > View > Property Panel.

To modify text:

  1. Select the text you want to modify. You can select individual characters, words, paragraphs, or the entire report.
  2. To format character font, size, weight, underlining, or color, on the Property Panel select an option under Text . For more information, see Editing Text.
  3. To set paragraph alignment, under Paragraph, select Align Left, Align Center, and Align Right and Justify..
  4. To add bullets or numbers a to selected paragraph, select the Bullet Style or Number Style under Paragraph.
  5. To remove bullets, click the Bullet Style or Number Style again.
