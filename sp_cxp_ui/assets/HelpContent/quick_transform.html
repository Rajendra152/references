Quick Transform

Use the Quick Transform dialog box to enter and execute simple, one-line mathematical functions to modify one or more columns of data.

Equation. To edit an existing transform, select it from the list and use the two text boxes to make changes to the right and left sides of the equation. You can also use the Functions that appear below. Select New Equation to create a new transform rather than edit an existing one. The check boxes to the right turn automatic updating on or off for individual transforms.

Important: For a complete list of all User-Defined Transforms, see Transform Function Descriptions.

Run button. This button initiates the transform.

Close button. Click this button to close the Quick Transform dialog box.

Options. Click to open the Optionsdialog box for transforms. Here you can select trigonometric units and the option to use the transform as the output column title is available.
