Equal Variance Test

Equal Variance test results display whether or not the data passed or failed the test of the assumption that the differences of the changes originate from a population with the same variance, and the P value calculated by the test. Equal variance of the source is assumed for all parametric tests.

This result appears if you enabled equal variance testing in the Options for Two Way RM ANOVA dialog box.
