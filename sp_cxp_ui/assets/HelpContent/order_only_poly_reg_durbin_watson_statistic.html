Durbin-Watson Statistic

The Durbin-Watson statistic is a measure of correlation between the residuals. If the residuals are not correlated, the Durbin-Watson statistic will be 2; the more this value differs from 2, the greater the likelihood that the residuals are correlated. This result appears if it was selected in the Options for Polynomial Regression dialog.
