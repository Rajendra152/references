Enabling Antialiased Lines on a Graph Page

Antialiasing is a technique used by computer graphic software to smooth out jagged, stair step lines. In SigmaPlot, you control this in the Options dialog box.
  1. Click the File tab.
  2. Click Options.
  3. In the Options dialog box, click the Page tab.
  4. Under AntiAliasing, select Enable AntiAliasing for page objects.
