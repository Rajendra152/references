Modifying Vector Plot Arrowheads

You can edit the degrees and length of vector plot arrowheads using the Graph Properties.

To edit the vector plot:

  1. Double-click a vector on a vector plot to open Graph Properties.
  2. Select Plot > Vectors from the Properties list.
  3. Use the Line Type options to modify how vector plot lines appear in the graph.
    1. To change the type of line, such as a dotted line or dashed, select an option from the Type drop-down list.
    2. To change the line color, select a color from the Color drop-down list.
    3. To change or add a gap color, select a color from the Gap Color drop-down list. This option is only available if you select a line type with actual "gaps" in it, like dotted or dashed, for example.
    4. To change the thickness of the line, enter a value in the Thickness box.
    5. To make the lines transparent, move the Transparency slider.
  4. Use the Arrowhead options to set arrowhead line type (lines or filled), angles in degrees, and the arrowhead lengths.
    1. To set the arrowhead line type, select Lines or Filled from the Type drop-down list.
    2. To set the angle of the arrowhead, move the Angle in degrees slider.
    3. To set the length of the arrowhead, you can select either:
      • Percentage of vector length. Enter a percentage into the field or move the slider.
      • Constant. Enter a value into the field or move the slider.
