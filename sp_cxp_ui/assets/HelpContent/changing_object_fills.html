Changing Object Fills

Change fill patterns and colors of drawn rectangles and ellipses, and of graph symbols, bars, and boxes using the Tools group on the Graph PageGraph tab

Note: When you select multiple objects, fill options apply to all selected objects that can be filled, including lines. Using Custom Colors and Incrementing Schemes

To change the background color of an object fill:

  1. Select the object(s) to modify on the graph page.
  2. In the Tools group on the Graph Page tab, click Fill Color.
