Fill color

You can create gradient area fills using the Color, Transparency, End Color, End Transparency, and Gradient Angle options.

Color. Change the fill color. This also sets the first basic color of the fill, if you choose to create a gradient fill. Or you can increment fill colors using the one of the incrementing schemes, or select a column of worksheet colors.

To define your own color, select (Custom), then define the color using the Color dialog box.
Note: Fill color automatically increments if you picked multiple XY column pairs.

To set an incremental color scheme, select (Color Schemes) and then click the ... to the right of the Color drop-down list. For more information, see Incrementing Color Schemes.

Transparency. This option corresponds with the Fill color selected above. Move the slider or enter a percentage.

End Color. To create a gradient fill, select an end color here.

End Transparency. Like Transparency above, this corresponds only with the End Color.
Tip: To make the area fill all one color and of the same level of transparency, simply make sure that the colors and percentages of the Color, Transparency, End Color, and End Transparency options math.

Gradient Angle. An angle of 0 degrees sets the gradient from top to bottom;

Pattern and edge

Color. Change the fill pattern and edge color. Increment pattern and edge colors using the one of the incrementing schemes. To define your own color, select (Custom), then define the color using the Color dialog box. Turn off pattern and edge colors by selecting (none).

Pattern. Change the fill pattern and density. Turn off pattern and edge colors by selecting (none).

Thickness. Change the thickness of the bar edge by moving the Thickness slider or by entering a thickness value.

Solid Bars. To make bars a solid color, set your Background color to the solid color you want to use, then select a pattern of (None) and set the Foreground color to either (None) or to the same color as the background.
