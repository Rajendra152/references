Shading in Different Directions

Use Graph Properties to change the direction of fill colors in an area plot.

To change the area fill direction:

  1. Create an area plot.
  2. Double-click the graph to open Graph Properties.
  3. Select Plot > Area Fills from the Properties list.
  4. From the Direction drop-down list, select Up, Down, Left, or Right.
