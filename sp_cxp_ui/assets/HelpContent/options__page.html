Options: Page

Page options control graph page properties.

Units. Sets the unit of measurement on the graph page (inches, millimeters, points).

Default zoom level. Set the default zoom level for all future graph pages.

Stretch maintains aspect ratio. When this box is selected, resized objects maintain their vertical-to-horizontal ratio when dragging a selected graph with corner handles. If this command is not checked, objects can be resized disproportionately.

Graph objects resize with graph. When this box is selected, resizing a graph automatically resizes objects associated with the graph, (axis labels, tick labels, the graph title, and the automatic legend). If this command is unchecked, objects must be sized individually.

Resolution independent patterns. Select this option when you want the pattern resolutions to print properly. If you are only going to view the graph on the screen, clear this option.

Show rulers. When this box is selected, horizontal and vertical rulers are displayed at the page window borders on the top and left-hand side of the page. If this option is clear, the rulers are hidden.

Show data values. When this box is selected, a tool tip appears with the X and Y data coordinates as you move the mouse pointer over the graph. If you don't want this tool tip to appear, clear this option; the coordinates still appear in the lower left corner of the SigmaPlot desktop.

Page undo. When this box is checked, the Undo and Redo commands are available for changes you make to the graph page. Disabling Undo and Redo can speed graph page operations significantly; however, page editing cannot be undone.

Show mini toolbars on selection. Select if you would like for mini toolbars to appear as you select objects on the page. If you would rather these mini toolbars not appear, then clear this option. You can still view the mini toolbar by right-clicking the object.

Grids.

  • Show grids. When this box is selected, grids are displayed on the graph page as either dots or lines in units as specified in the Units scroll-down list. If this option is clear, the grids remain hidden.
  • Grid density. Sets the grid density for both x and y grid directions. You can choose a density value from the drop-down list, or enter any legitimate value.
  • Snap-to. When this box is selected, all drawn, resized, or moved objects "snap" to the nearest grid point. When drawing or resizing, the current corner or edge being dragged is snapped. When moving an object, the upper left corner is snapped.

3D.

  • Transparent bitmaps. Select to place a transparent bitmap of your 3D graph, versus an opaque one, onto the page, so you can see other 2D graphs and objects behind it.
  • Export resolution. As transparent bitmaps can take longer to print and copy and paste, you can set the resolution here to a smaller (or larger) bitmap. Just be aware that a lower resolution bitmap will be of lesser quality in appearance.
AntiAliased.

Antialiasing is a technique used by computer graphic software to smooth out jagged, stair step lines.

  • Enable AntiAliasing for page objects. Select this option to smooth out jagged edges on curved lines on a graph.
Printing.
  • Print graph page as bitmap.
