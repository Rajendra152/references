Setting Pearson Correlation Options

Use the Pearson Correlation options to:

  • Set the assumption checking options for normality
  • Set the display format for results in reports.

To change Pearson Correlation options:

  1. If you are going to run the test after changing test options and want to select your data before you run the test, drag the pointer over your data.
  2. Select Pearson Correlation from the Select Test drop-down list in the SigmaStat group on the Analysis tab.
  3. Click Options. The Options for Pearson Correlation dialog box appears with two tabs:
      • Assumption Checking. Click the Assumption Checking tab to view the Normality options.
      • Results.Click the Results tab to view formatting options for reports

    Options settings are saved between SigmaPlot sessions.

  4. To continue the test, click Run Test.
  5. To accept the current settings and close the dialog box, click OK.
