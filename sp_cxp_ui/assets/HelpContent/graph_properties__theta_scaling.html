Graph Properties: Theta Scaling

Graph. Select the current graph to edit. All graphs on the page are listed here.

Axis. Select the current axis to edit. All axes for the current plot are listed here.

Plot. Select the current plot to edit. All plots for the current graph are listed here.

Scale type. Select a scale type.

Angular units. Set the measurement units of the angular axis to one of the following:
Unit Default Range
Degrees 0-360
Radians 0-2pi
Grads 400
Months 0-13
Weeks 0-53
Days 8
Hours 0-24
Minutes/Seconds 0-60
Other Custom
Note: The only effect of changing units is to change the pre-defined axis range. The range can be manually changed regardless of the current units.

Range

From. Set the starting range for the angular axis.

To. Set the ending range for the angular axis.

Note: This value will occur at the same location as Scale Type Set the axis scale. The default axis scale is Linear for all numeric data, Category for text data, and Date/Time for date and time data. Linear Common Log Natural Log Probability Probit Logit Category Date/Time
Arc. Use the slider control to set the displayed arc of the polar plot, or enter a value into the edit box. A setting of 360° displays the entire circle, 270° displays three-quarters of the circle, 90° displays half of the circle, etc.
Note: Changing the arc of the angular axis does not affect the axis range. The current axis range is displayed along the new distance of the arc.

Start angle. Use the slider control to rotate the polar plot. The default is 0° (3 o’clock). Rotation is always counterclockwise.
