Graph Page Tools

Use the Tools group on the Graph Page tab to draw objects on the graph page.

The drawing tools on the Tools group are:

Select Object

Use the Select Object button to select objects on the graph page.
Text

Click this button to add text, labels, or manually created legends to the graph page.
Draw Line

Click this button to draw a line on the graph page.
Draw Arrow

Click this button to draw an arrow on the graph page.
Draw Box

Use the Draw Box button to draw a box on the graph page.
Draw Ellipse

Click this button to draw an ellipse on the graph page.
