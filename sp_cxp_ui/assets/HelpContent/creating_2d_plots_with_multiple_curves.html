Creating 2D Plots with Multiple Curves

You do not have to create multiple plots to obtain multiple curves. To plot more than one curve, choose any of the plot styles described as Multiple and add additional columns, or column pairs to the list of curves in the Graph Wizard. Both the Graph Wizard and the graph styles available on the Create Graph tab list all types of multiple plot graph styles.

The order of the curves is determined by the order of the column pairs in the Graph Wizard. To change the curve order, select columns again in the Graph Wizard or select the column in the worksheet.
