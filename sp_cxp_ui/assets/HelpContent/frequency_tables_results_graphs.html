Frequency Tables Results Graphs

You can generate a frequency bar chart from a one way frequency table report.

A frequency bar chart is a bar chart with category names on the x-axis and the contribution of each category to the data set shown on the y-axis.

You can scale the y-axis as either Frequency or Percent. The dialog box will also contain a combo box for selecting the data column that will be represented in the graph. An example of this graph is shown below. There may be a limit on the number of categories that can be represented.
