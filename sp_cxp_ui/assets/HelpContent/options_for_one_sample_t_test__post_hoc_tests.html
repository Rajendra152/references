Options for One-Sample t-Test: Post Hoc Tests

Power. The power or sensitivity of a test is the probability that the test will detect a difference between the groups if there is really a difference.

Use Alpha Value. Alpha (α) is the acceptable probability of incorrectly concluding that there is a difference. The suggested value is α = 0.05. This indicates that a one in twenty chance of error is acceptable, or that you are willing to conclude there is a significant difference when P < 0.05.

Smaller values of α result in stricter requirements before concluding there is a significant difference, but a greater possibility of concluding there is no difference when one exists. Larger values of α make it easier to conclude that there is a difference, but also increase the risk of reporting a false positive.
