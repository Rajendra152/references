How do you want to specify the independent variables?

If you chose to predict a dependent variable from several independent variables, you can select the independent variables using two methods. The dependent variable and independent variables are selected as columns from the worksheet when the regression procedure is performed.

Select one of the following:

  • Include all selected independent variables in the equation. Select this option if you want to compute a single equation using all independent variables you select for the equation, regardless of whether they contribute significantly to predicting the dependent variable.

    If you select this option, click Finish. SigmaPlot suggests performing a Multiple Linear Regression. See Multiple Linear Regression. You can also read descriptions of the results for this procedure in Interpreting Multiple Linear Regression Results.

  • Let SigmaPlot select the "best" variables to include in the equation. Select this option if you want SigmaPlot to screen the potential independent variables you select and only include ones that significantly contribute to predicting the dependent variable. You are then asked how you want to select the independent variables. See How do you want SigmaPlot to select the independent variable?.
