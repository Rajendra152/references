Histogram Options

Automatic binning. Selected by default. The algorithm used for automatic binning calculates the number of bins based upon the number of data points and the skewness of the data. where N = number of non-missing points, is the adjusted Fisher-Pearson standardized moment coefficient, and is the standard deviation of

To specify a different number of bins, clear Automatic binning and select a number from the Number of bins list. You can enter values from 1 to 100.

Normalization. The Normalization drop-down list gives three options for scaling the histogram values. If you select None, then bin height is the count or frequency of the data in the bin. If you select Fraction, then bin height is the fraction of the total data count assigned to the bin. If you select Percent, then bin height is the percentage of the total data count assigned to the bin.

Bin type. The Bin Type buttons provide two options to determine how to assign data that lies between two bins.
  • If you select Left-edge binning, then any data that falls on the left edge of any bin is assigned to that bin. This is selected by default.
  • If you select Right-edge binning, then any data that falls on the right edge of any bin is assigned to that bin.
