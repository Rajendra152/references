Axis Scale Types

Scale types appear under Axis in the Properties list in Graph Properties.

  1. Double-click an axis on a graph to open Graph Properties.
  2. Select Axis > Scaling from the Properties list.
    The available scale types appear in the Type list. They include:
    • Linear
    • Common Log
    • Natural Log
    • Probability
    • Probit
    • Logit
    • Category
    • Date/time
    • Reciprocal
    • Weibull
