Three Way Effects

Three Way Effects graphs are available for the following test:

  • Three Way Analysis of Variance (ANOVA).

For Three Way Effects in Three Way ANOVA, the number of graphs equals the number of levels of the third factor (which is the last factor that was selected for running the test). Each graph for Three Way Effects contains multiple profile plots, one for each level of one of the second factor (which is the factor that was selected second for running the test).
