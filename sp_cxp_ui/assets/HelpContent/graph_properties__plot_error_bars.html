Axis. Select the current axis to edit. All axes for the current plot are listed here.

Plot. Select the current plot to edit. All plots for the current graph are listed here.

Error bars

Control the logic for error bar directions.

Direction. Control how error bar directions are determined.

  • Absolute. Always point error bars in the selected directions.

  • X positive. Error bars always point to the right. This option is dimmed if there are no horizontal error bars.

  • Y positive. Error bars always point up. This option is dimmed if there are no vertical error bars.

  • X negative. Error bars always point to the left. This option is dimmed if there are no horizontal error bars.

  • Y negative. Error bars always point down. This option is dimmed if there are no vertical error bars.

  • Relative. Point error bars toward or away from zero. This option is useful for bar charts that have negative values. You can also enter error bar directions into a worksheet column, then select this column from the drop down list.

  • X from zero. Error bars always away from zero. This option is dimmed if there are no horizontal error bars.

  • Y from zero. Error bars always point away from zero. This option is dimmed if there are no vertical error bars. X To Zero Error bars always points toward zero. This option is dimmed if there are no horizontal error bars.

  • Y to zero. Error bars always point toward zero. This option is dimmed if there are no vertical error bars.

Line

Set the line properties of the error bars Color list Select a color to use for error bars.

Color. Select a color to use for error bars.

To set an incremental color scheme for the error bars, select (Color Schemes) and then click the ... to the right of the Color drop-down list. For more information, see Incrementing Color Schemes.

Thickness. Move the slider control to change error bar line thickness, or type a new value into the edit box.

Cap width. Use the slider control to change line error bar cap width, or type a new value into the edit box.

Mean computation. Set the error bar computations, if applicable.
