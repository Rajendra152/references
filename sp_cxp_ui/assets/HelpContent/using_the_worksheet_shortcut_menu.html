Using the Worksheet Shortcut Menu

In addition to the commands available in SigmaPlot ribbons, right-clicking the worksheet displays a shortcut menu. The commands on the right-click shortcut menu include the Cut, Copy, Paste, Transpose Paste, Insert Cells and Delete Cells commands.
