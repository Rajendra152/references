Contour Plots

Contour

Plots data XYZ values in 2D space. Format data columns as:

  • XYZ Triplet
  • Many Z
  • XY, Many Z
Filled Contour

Plots data XYZ values in 2D space filling in the area between contour levels. Format data columns as:

  • XYZ Triplet
  • Many Z
  • XY, Many Z
