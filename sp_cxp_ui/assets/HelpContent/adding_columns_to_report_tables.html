Adding Columns to Report Tables

  1. Click on a cell in a table to the left or to the right of where you'd like the new column to appear.
  2. Right-click to reveal the shortcut menu.
  3. To add a column to the right of the cell, select Insert > Insert Right.
  4. To add a column to the left of the cell, select Insert > Insert Left.
