Performing a Paired t-test

To perform a Paired t-test:

  1. Enter or arrange your data in the worksheet.
  2. If desired, set the Paired t-test options.
  3. Click the Analysis tab.
  4. In the SigmaStat group, from the Tests drop-down list, select:
    Before and After > Paired t-test
  5. Run the test.
  6. Generate report graphs.
