Normality Report Graphs

You can generate two graphs using the results from a Normality report. They include a:

  • Histogram of the residuals. The Normality histogram plots the raw residuals in a specified range, using a defined interval set.
  • Normal probability plot of the residuals. The Normality probability plot graphs the frequency of the raw residuals.
