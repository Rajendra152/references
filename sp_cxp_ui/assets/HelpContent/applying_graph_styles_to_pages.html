Applying Graph Styles to Pages

Use the Graph Style Gallery to quickly apply your own custom graph styles to data.

To apply a graph style:

  1. View the Graph Style Gallery.
  2. Double-click the graph style you want to use.
    The Graph Wizard - Create Graph panel appears.
  3. In the Graph Wizard, select the worksheet columns you want to use for the plot.
  4. Click Finish to create the plot.
