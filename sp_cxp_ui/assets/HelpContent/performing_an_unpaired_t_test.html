Performing an Unpaired t-Test

To perform an Unpaired t-test:

  1. Enter or arrange your data appropriately in the worksheet.
  2. If desired, set the t-test options.
  3. Click the Analysis tab.
  4. In the SigmaStat group, from the Tests drop-down list, select:
    Compare Two Groups > t-test
  5. Run the test.
  6. Generate report graphs.
