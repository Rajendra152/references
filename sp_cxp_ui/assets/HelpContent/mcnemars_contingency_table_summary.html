Contingency Table Summary

Each cell in the table is described with a set of statistics for that cell.

Observed Counts. These are the number of observations per cell, obtained from the contingency table data.

Expected Frequencies. The expected frequencies for each cell in the contingency table, as predicted using the row and columns percentages.
